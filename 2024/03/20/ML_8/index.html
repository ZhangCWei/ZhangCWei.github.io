贝叶斯决策理论

后验概率

  • 给定某观测到的特征x\boldsymbol{x}x，其属于特定类别的概率P(y∣x)P(y|\boldsymbol{x})P(y∣x)

贝叶斯定理

P(y∣x)=P(x∣y)×P(y)p(x)=P(x∣y)P(y)∑yp(x,y)P(y|\boldsymbol{x}) = \frac{P(\boldsymbol{x}|y)×P(y)}{p(\boldsymbol{x})} = \frac{P(\boldsymbol{x}|y)P(y)}{\sum_yp(\boldsymbol{x},y)} P(y∣x)=p(x)P(x∣y)×P(y)​=∑y​p(x,y)P(x∣y)P(y)​

  • 似然函数：P(x∣y)P(\boldsymbol{x}|y)P(x∣y)
  • 先验概率：P(y)P(y)P(y)
  • 后验概率：P(y∣x)P(y|\boldsymbol{x})P(y∣x)
  • 归一化因子：p(x)p(\boldsymbol{x})p(x)

最大化后验概率

  • 取最大化后验概率的类别作为分类结果。

y∗=argmaxy P(y∣x)y^* = \underset{y}{argmax}\ P(y|\boldsymbol{x}) y∗=yargmax​ P(y∣x)

朴素贝叶斯

公式推导

P(y∣x)=P(x∣y)×P(y)p(x)P(y|\boldsymbol{x}) = \frac{P(\boldsymbol{x}|y)×P(y)}{p(\boldsymbol{x})} P(y∣x)=p(x)P(x∣y)×P(y)​

  • 其中，所有属性联合概率在有限样本下难以获取：

P(x∣y)=P(x1,x2,...,xd∣y)=P(x1∣y)P(x2∣x1,y)...P(xd∣x1,x2,...,xd−1,y)P(\boldsymbol{x}|y) = P(x_1,x_2,...,x_d|y) = P(x_1|y)P(x_2|x_1,y)...P(x_d|x_1,x_2,...,x_{d-1},y) P(x∣y)=P(x1​,x2​,...,xd​∣y)=P(x1​∣y)P(x2​∣x1​,y)...P(xd​∣x1​,x2​,...,xd−1​,y)

  • 属性条件独立性假设：所有属性相互独立

P(x∣y)=P(x1∣y)P(x2∣y)...P(xd∣y)P(\boldsymbol{x}|y) = P(x_1|y)P(x_2|y)...P(x_d|y) P(x∣y)=P(x1​∣y)P(x2​∣y)...P(xd​∣y)

  • 即：

P(x1,x2,...,xd∣y)=∏i=1dP(xi∣y)P(x_1,x_2,...,x_d|y) = \prod_{i=1}^dP(x_i|y) P(x1​,x2​,...,xd​∣y)=i=1∏d​P(xi​∣y)

  • 因此，在属性条件独立性假设下，公式可以转化为：

P(y∣x)=∏i=1dP(xi∣y)P(y)p(x)P(y|\boldsymbol{x}) = \frac{\prod_{i=1}^dP(x_i|y)P(y)}{p(\boldsymbol{x})} P(y∣x)=p(x)∏i=1d​P(xi​∣y)P(y)​

EM算法

问题介绍

  • 存在隐变量(即未观测到的属性)时的数据分布参数估计
  • 示例：
    • 学生身高分布估计：在一个班级中，假设男生身高服从高斯分布𝑁1(𝑢1,𝜎12)𝑁_1(𝑢_1, 𝜎_1^2)N1​(u1​,𝜎12​)，女生身高服从另一高斯分布𝑁2(𝑢2,𝜎22)𝑁_2(𝑢_2, 𝜎_2^2)N2​(u2​,𝜎22​) ，已知班上所有同学的身高数据，但不知道各个学生的性别，估计这两个高斯分布
    • 零件长度分布估计：在一堆零件中，假设好质量零件长度服从高斯分布𝑁1(𝑢1,𝜎12)𝑁_1(𝑢_1, 𝜎_1^2)N1​(u1​,𝜎12​)，坏质量零件长度服从另一高斯分布𝑁2(𝑢2,𝜎22)𝑁_2(𝑢_2, 𝜎_2^2)N2​(u2​,𝜎22​) ，已知所有零件长度数据，但尚未知道各个零件的质量好坏，估计这两个高斯分布
  • 存在未知的隐变量𝑧，即学生的性别、零件的质量
  • 最大似然法求解：maxθ logP(X∣θ)\underset{\theta}{max}\ logP(\boldsymbol{X}|\theta)θmax​ logP(X∣θ)

基本思想

  • E步 (Expectation)：当参数𝜃已知，根据训练数据推断出最优隐变量𝑧的值
  • M步 (Maximization)：当𝑧已知，对𝜃做极大似然估计
  • 公式：maxθ Ez[log P(x,z∣θ)]\underset{\theta}{max}\ E_z[log\ P(\boldsymbol{x},z|\theta)]θmax​ Ez​[log P(x,z∣θ)]

  • 初始化𝜃0𝜃_0𝜃0​
  • 迭代执行以下步骤直至收敛： 𝑡 = 0,1,2, …
    • E 步：基于𝜃𝑡𝜃_𝑡𝜃t​，推断隐变量𝑧的分布𝑃(𝑧∣𝑿,𝜃𝑡)𝑃(𝑧|𝑿,𝜃_𝑡)P(z∣X,𝜃t​)，并计算对数似然函数在该分布下的期望，即𝑄(θ)=𝐸𝑧∼𝑃(𝑧∣𝑿,θ𝑡) log 𝑃(𝑿,𝑧∣θ)𝑄(\theta)=𝐸_{𝑧\sim𝑃(𝑧|𝑿,\theta_𝑡)}\ log\ 𝑃(𝑿,𝑧|\theta)Q(θ)=Ez∼P(z∣X,θt​)​ log P(X,z∣θ)
