机器学习10：神经网络

发表于 2024-03-28 更新于 2024-04-15 分类于 ML
神经网络是一种运算模型，由大量的节点（或称神经元）之间相互联接构成。每个节点代表一种特定的输出函数，称为激励函数。每两个节点间的连接都代表一个对于通过该连接信号的加权值，称之为权重，这相当于人工神经网络的记忆。网络的输出则依网络的连接方式，权重值和激励函数的不同而不同。而网络自身通常都是对自然界某种算法或者函数的逼近，也可能是对一种逻辑策略的表达...

神经网络

背景介绍

  • 神经网络是由具有适应性的简单单元组成的广泛并行互联的网络, 它的组织能够模拟生物神经系统对真实世界物体所作出的反应。
  • 生物神经网络
    • 每个神经元与其他神经元相连, 当它“兴奋” 时, 就会向相连的神经元发送化学物质, 从而改变这些神经元内的电位
    • 如果某神经元的电位超过一个“阈值” , 那么它就会被激活, 即“兴奋” 起来, 向其它神经元发送化学物质
  • 神经元模型
    • M-P神经元模型(McCulloch and Pitts)
      • 输入：来自其他神经元传递过来的输入信号
      • 处理：输入信号通过带权重的连接进行传递, 神经元接受到总输入值将其与神经元的阈值进行比较
      • 输出：通过激活函数的处理以得到输出
  • 激活函数
    • 逻辑回归的建模可视为一种比较简单的神经元，其激活函数为Sigmoid

神经网络

  • 基础概念
    • 多个神经元相互连接
    • 表征特征(𝒙)到标记(y)的映射
  • 广泛应用
  • 神经网络和深度学习的发展因素

    • 算力：GPU

  • 激活函数
    • 激活函数要求
      • 非线性、简单
      • 连续并可导（允许少数点上不可导）
      • 导数的值域在一个合适的区间，不能太大也不能太小
    • 激活函数选择
      • 通常情况下，默认选择ReLU；tanh也行
      • 输出是0,1值（二分类问题），输出层选sigmoid，其他层选ReLU
  • - Sigmoid激活函数 - 优点 - 梯度的“平滑性” - 输出在“ 0 - 1 区间” - 缺点 - 梯度消失问题： 神经网络使用 Sigmoid 激活函数进行反向传播时，输出接近 0 或 1 的神经元其梯度趋近于 0 - 计算成本问题：涉及指数计算 - 不以零为中心： Sigmoid 输出不以零为中心
  • - tanh激活函数 - 优点 - 梯度的“平滑性” - 输出以零为中心 - 缺点 - **梯度消失**问题： 神经网络使用 tanh 激活函数进行**反向传播**时，输出接近 -1 或 1 的神经元其梯度趋近于 0 - 计算成本问题：涉及指数计算
  • - ReLU激活函数 - 优点 - 计算效率较高 - 兼具线性和非线性特性 - 缺点 - 梯度消失问题： 在 𝑧 < 0 时， 神经元保持**非激活**状态，且在反向传导(backward pass)过程中梯度为零
  • softmax输出
    • 考虑多分类问题：若标记有 𝐶 个类别 𝑦 ∈ {1,2, … 𝐶}，即𝐶分类问题，则：
