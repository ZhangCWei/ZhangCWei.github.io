马尔可夫过程

马尔可夫过程概述

  • 智能体 (agent) 学习在环境 (environment) 中采取动作 (action) 以最大化奖励 (reward) 的策略 (policy)
  • 马尔可夫决策过程 (Markov decision process, MDP) 描述了强化学习的环境规律， 几乎所有的强化学习问题都可以形式化为MDPs
  • 马尔可夫性质
    • P[St+1∣St]=P[St+1∣S1,S2,...,St]P[S_{t+1}|S_t] = P[S_{t+1}|S_1,S_2,...,S_t]P[St+1​∣St​]=P[St+1​∣S1​,S2​,...,St​]
    • 给定当前状态𝑆𝑡𝑆_𝑡St​ ，未来𝑆𝑡+1𝑆_{𝑡+1}St+1​与历史𝑆1,𝑆2,𝑆3,…𝑆_1, 𝑆_2, 𝑆_3,…S1​,S2​,S3​,…无关
    • 𝑆𝑡+1𝑆_{𝑡+1}St+1​取决于𝑆𝑡𝑆_𝑡St​，不用考虑历史状态
  • 状态转移矩阵
    • 当前状态跳往下一个状态的概率：$ 𝑃[𝑆_{𝑡+1}=𝑠’|𝑆_𝑡 = 𝑠]$
    • 状态转移矩阵PPP：

P=[P(s1∣s1)P(s2∣s1)...P(sN∣s1)P(s1∣s2)P(s2∣s2)...P(sN∣s2)⋮⋮⋱⋮P(s1∣sN)P(s2∣sN)...P(sN∣sN)]P=\begin{bmatrix}P(s_1|s_1)&P(s_2|s_1)&...&P(s_N|s_1)\\P(s_1|s_2)&P(s_2|s_2)&...&P(s_N|s_2)\\\vdots&\vdots&\ddots&\vdots\\P(s_1|s_N)&P(s_2|s_N)&...&P(s_N|s_N)\end{bmatrix} P=​P(s1​∣s1​)P(s1​∣s2​)⋮P(s1​∣sN​)​P(s2​∣s1​)P(s2​∣s2​)⋮P(s2​∣sN​)​......⋱...​P(sN​∣s1​)P(sN​∣s2​)⋮P(sN​∣sN​)​​

  • 马尔可夫过程是满足马尔可夫性质的离散随机过程，又称马尔可夫链

    • 有限个状态集合𝑆

    • 状态转移矩阵𝑃

马尔可夫奖赏过程

  • 马尔可夫奖赏过程：马尔可夫链+奖赏

    • 有限个状态集合 𝑆
    • 状态转移矩阵 𝑃
    • **奖赏函数 **𝑅(𝑠)=𝔼[𝑅𝑡+1∣𝑆𝑡=𝑠]𝑅(𝑠) = 𝔼[𝑅_{𝑡+1}|𝑆_𝑡 = 𝑠]R(s)=E[Rt+1​∣St​=s]
    • 折扣因子 𝛾∈[0,1]𝛾 ∈ [0,1]𝛾∈[0,1]

  • 回报

    • 从时间𝑡开始，获得的总折扣奖赏
    • 折扣因子𝛾：表征未来的奖赏在当前时刻的折现

𝐺𝑡=𝑅𝑡+1+𝛾𝑅𝑡+2+𝛾2𝑅𝑡+3+⋯=∑𝑘=0∞𝛾𝑘𝑅𝑡+𝑘+1𝐺_𝑡 = 𝑅_{𝑡+1} + 𝛾𝑅_{𝑡+2} + 𝛾^2𝑅_{𝑡+3} + ⋯ = \sum_{𝑘=0}^∞ 𝛾^𝑘𝑅_{𝑡+𝑘+1} Gt​=Rt+1​+𝛾Rt+2​+𝛾2Rt+3​+⋯=k=0∑∞​𝛾kRt+k+1​

  • 若时间段有限

𝐺𝑡=𝑅𝑡+1+𝛾𝑅𝑡+2+𝛾2𝑅𝑡+3+𝛾T−t−1𝑅T𝐺_𝑡 = 𝑅_{𝑡+1} + 𝛾𝑅_{𝑡+2} + 𝛾^2𝑅_{𝑡+3} + 𝛾^{T-t-1}𝑅_{T} Gt​=Rt+1​+𝛾Rt+2​+𝛾2Rt+3​+𝛾T−t−1RT​

  • 示例（𝛾 = 0.5）
    • 两条可能的状态轨迹：
      • 𝑆1𝑆_1S1​ = 学习，𝑆2𝑆_2S2​ = 吃饭，𝑆3𝑆_3S3​ = 娱乐，𝑆4𝑆_4S4​ = 睡觉
        • $𝐺_1 = 2 + 1 × 0.5 + 2 × 0.5^2 + 1 × 0.5^3 $
      • 𝑆1𝑆_1S1​= 吃饭，𝑆2𝑆_2S2​ = 睡觉，𝑆3𝑆_3S3​ = 娱乐，𝑆4𝑆_4S4​ = 学习，𝑆5𝑆_5S5​ = 睡觉
      • $𝐺_1 = 1 + 1 × 0.5 + 2 × 0.5^2 + 2 × 0.5^3 + 1 × 0.5^4 $

  • 折扣因子

    • 避免在计算无限时间段下的回报出现无限大的情况
    • 未来的奖赏具有不确定性
    • 未来的钱不值钱（利息）
    • 个体更偏向于当前奖赏
      • 𝛾 = 0: 只关注当前奖赏
      • 𝛾 = 1: 未来奖赏和当前奖赏同样重要

  • 状态值函数

    • 从状态𝑠开始，获得的回报的期望
    • 状态值函数表征该状态的长期价值

𝑉𝑠=𝔼[𝐺𝑡∣𝑆𝑡=𝑠]=𝔼[𝑅𝑡+1+𝛾𝑅𝑡+2+𝛾2𝑅𝑡+3+⋯∣𝑆𝑡=𝑠]𝑉_𝑠 = 𝔼[𝐺_𝑡|𝑆_𝑡=𝑠]=𝔼[𝑅_{𝑡+1}+𝛾𝑅_{𝑡+2}+𝛾^2𝑅_{𝑡+3}+⋯|𝑆_{𝑡}=𝑠] Vs​=E[Gt​∣St​=s]=E[Rt+1​+𝛾Rt+2​+𝛾2Rt+3​+⋯∣St​=s]

  • 贝尔曼方程
    • 价值函数可分解为两个部分
      • 立即奖赏𝑅𝑡+1𝑅_{𝑡+1}Rt+1​
      • 折扣的后续状态的价值函数𝛾𝑉(𝑆𝑡+1)𝛾𝑉(𝑆_{𝑡+1})𝛾V(St+1​)

V(s)=E[Rt+1+γRt+2+γ2Rt+3+⋯∣St=s]=E[Rt+1+γ(Rt+2+γRt+3+⋯ )∣St=s]=E[Rt+1+γGt+1∣St=s]=E[Rt+1+γV(St+1)∣St=s]\begin{align} V(s) &= \mathbb{E}[R_{t+1} + \gamma R_{t+2} + \gamma^2 R_{t+3} + \dotsb | S_t = s] \\ &= \mathbb{E}[R_{t+1} + \gamma(R_{t+2} + \gamma R_{t+3} + \dotsb) | S_t = s] \\ &= \mathbb{E}[R_{t+1} + \gamma G_{t+1} | S_t = s] \\ &= \mathbb{E}[R_{t+1} + \gamma V(S_{t+1}) | S_t = s] \end{align} V(s)​=E[Rt+1​+γRt+2​+γ2Rt+3​+⋯∣St​=s]=E[Rt+1​+γ(Rt+2​+γRt+3​+⋯)∣St​=s]=E[Rt+1​+γGt+1​∣St​=s]=E[Rt+1​+γV(St+1​)∣St​=s]​​

  • 矩阵形式
    • V=R+γPVV = R + \gamma PVV=R+γPV
    • 解析解：𝑉=(𝐼−𝛾𝑃)−1𝑅𝑉 = (𝐼-𝛾𝑃)^{-1}𝑅V=(I−𝛾P)−1R

[V(s1)V(s2)⋮V(sN)]=[R(s1)R(s2)⋮R(sN)]+γ[P(s1∣s1)P(s2∣s1)...P(sN∣s1)P(s1∣s2)P(s2∣s2)...P(sN∣s2)⋮⋮⋮⋮P(s1∣sN)P(s2∣sN)...P(sN∣sN)][V(s1)V(s2)⋮V(sN)]\begin{bmatrix}V(s_1)\\V(s_2)\\\vdots\\V(s_N)\end{bmatrix}=\begin{bmatrix}R(s_1)\\R(s_2)\\\vdots\\R(s_N)\end{bmatrix}+\gamma\begin{bmatrix}P(s_1|s_1)&P(s_2|s_1)&...&P(s_N|s_1)\\P(s_1|s_2)&P(s_2|s_2)&...&P(s_N|s_2)\\\vdots&\vdots&\vdots&\vdots\\P(s_1|s_N)&P(s_2|s_N)&...&P(s_N|s_N)\end{bmatrix}\begin{bmatrix}V(s_1)\\V(s_2)\\\vdots\\V(s_N)\end{bmatrix} ​V(s1​)V(s2​)⋮V(sN​)​​=​R(s1​)R(s2​)⋮R(sN​)​​+γ​P(s1​∣s1​)P(s1​∣s2​)⋮P(s1​∣sN​)​P(s2​∣s1​)P(s2​∣s2​)⋮P(s2​∣sN​)​......⋮...​P(sN​∣s1​)P(sN​∣s2​)⋮P(sN​∣sN​)​​​V(s1​)V(s2​)⋮V(sN​)​​

  • 复杂度𝑂(𝑁^3^)，适用于规模较小的 MRP
  • 迭代法求解大规模 MRP：动态规划、蒙特卡洛、时间差分

马尔可夫决策过程

  • 马尔可夫决策过程：马尔可夫奖赏过程+决策
    • 有限个状态集合 𝑆
    • 有限个动作集合 𝐴
    • 状态转移矩阵𝑃: 𝑃[𝑆𝑡+1=𝑠′∣𝑆𝑡=𝑠,𝐴𝑡=𝑎]𝑃:\ 𝑃[𝑆_{𝑡+1}=𝑠'|𝑆_𝑡=𝑠,𝐴_𝑡 = 𝑎]P: P[St+1​=s′∣St​=s,At​=a]
    • 奖赏函数𝑅(𝑠,𝑎)=𝔼[𝑅𝑡+1∣𝑆𝑡=𝑠,𝐴𝑡=𝑎]𝑅(𝑠, 𝑎) = 𝔼[𝑅_{𝑡+1}|𝑆_𝑡=𝑠,𝐴_𝑡=𝑎]R(s,a)=E[Rt+1​∣St​=s,At​=a]
    • 折扣因子𝛾∈[0,1]𝛾 ∈ [0,1]𝛾∈[0,1]

  • 策略

    • 表征了在每个状态的采取什么动作
    • 𝜋(𝑎|𝑠) ：当前状态为 𝑠 的时候，采取动作 𝑎 的概率
    • 策略只由当前的状态决定
    • 平稳策略：与时间无关，即对任意𝑡，均有𝐴~𝑡~~𝜋(𝑎|𝑠)

  • 价值函数

    • 状态值函数

      • 𝑉𝜋(𝑠)=𝔼𝜋[𝐺𝑡∣𝑆𝑡=𝑠]=𝔼𝜋[𝑅𝑡+1+𝛾𝑅𝑡+2+𝛾2𝑅𝑡+3+⋯∣𝑆𝑡=𝑠]𝑉^𝜋(𝑠)=𝔼_𝜋[𝐺_𝑡|𝑆_𝑡=𝑠]=𝔼_𝜋[𝑅_{𝑡+1}+𝛾𝑅_{𝑡+2}+𝛾^2𝑅_{𝑡+3}+⋯|𝑆_𝑡 = 𝑠]V𝜋(s)=E𝜋​[Gt​∣St​=s]=E𝜋​[Rt+1​+𝛾Rt+2​+𝛾2Rt+3​+⋯∣St​=s]
    • 动作值函数
      • 从状态 𝑠 执行动作 𝑎 开始，采取策略 𝜋，获得的回报的期望
      • \begin{align}𝑞^𝜋(𝑠,𝑎) &= 𝔼_𝜋[𝐺_𝑡|𝑆_𝑡=𝑠,𝐴_𝑡=𝑎]\\ &= 𝔼_𝜋[𝑅_{𝑡+1}+𝛾𝑅_{𝑡+2}+𝛾^2𝑅_{𝑡+3}+⋯|𝑆_𝑡=𝑠,𝐴_𝑡=𝑎]\end{align}
