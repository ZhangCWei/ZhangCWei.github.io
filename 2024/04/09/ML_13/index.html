机器学习13：无模型预测与控制

发表于 2024-04-09 更新于 2024-04-15 分类于 ML
在已知的马尔可夫决策过程中，无论是策略迭代还是价值迭代，都假定已知环境的动态和奖励，然而在许多的真实世界的问题中，MDP模型或者是未知的，或者是已知的但计算太复杂。本文讲述无模型的预测与控制（Model-free Prediction and Control），通过与环境的交互迭代来求解问题...

无模型预测与控制

  • 在很多实际应用中，马尔可夫决策过程模型（包括奖赏和状态转移矩阵）往往难以获知或者规模太大难以直接使用

    • 例子：游戏、围棋、股票投资

  • 解决方案：无模型（model-free）强化学习

    • 不假设模型（包括奖赏和状态转移矩阵）已知
    • 通过与环境进行交互来进行学习

  • 无模型预测： 评估给定的策略

    • 输出： 状态价值函数𝑉𝜋(𝑠)𝑉^𝜋(𝑠)V𝜋(s)

    • 方法：蒙特卡罗法和时间差分法

  • 无模型控制： 寻找最优的策略

    • 输出： 最优动作价值函数𝑞(𝑠,𝑎)𝑞(𝑠,𝑎)q(s,a)和最优策略
    • 方法：蒙特卡罗法和时间差分法

时间差分法

  • 时间差分 (Temporal-Difference, TD)
    • 无模型方法
    • 在回合过程中进行学习，无需等待回合结束
    • 自举 (bootstrapping)
  • 增量MC：往实际采样到的回报𝐺𝑡𝐺_𝑡Gt​进行更新

𝑉(𝑆𝑡)←𝑉(𝑆𝑡)+α(𝐺𝑡−𝑉(𝑆𝑡))𝑉(𝑆_𝑡) ← 𝑉(𝑆_𝑡) + \alpha(𝐺_𝑡-𝑉(𝑆_𝑡)) V(St​)←V(St​)+α(Gt​−V(St​))

  • 贝尔曼方程：价值函数等于立即奖赏 + 折扣的后续状态价值函数

𝑉(𝑆𝑡)=𝔼[𝐺𝑡∣𝑆𝑡]=𝔼[𝑅𝑡+1+𝛾𝑉(𝑆𝑡+1)∣𝑆𝑡]𝑉(𝑆_𝑡) = 𝔼[𝐺_𝑡|𝑆_𝑡] = 𝔼[𝑅_{𝑡+1} + 𝛾𝑉(𝑆_{𝑡+1})|𝑆_𝑡] V(St​)=E[Gt​∣St​]=E[Rt+1​+𝛾V(St+1​)∣St​]

  • 时间差分 (TD)： 往𝑅𝑡+1+𝛾𝑉(𝑆𝑡+1)𝑅_{𝑡+1} + 𝛾𝑉(𝑆_{𝑡+1})Rt+1​+𝛾V(St+1​)进行更新，其中：
    • 𝑅𝑡+1+𝛾𝑉(𝑆𝑡+1)𝑅_{𝑡+1} + 𝛾𝑉(𝑆_{𝑡+1})Rt+1​+𝛾V(St+1​)：TD target（目标）
    • 𝑅𝑡+1+𝛾𝑉(𝑆𝑡+1)−𝑉(𝑆𝑡)𝑅_{𝑡+1} + 𝛾𝑉(𝑆_{𝑡+1})-𝑉(𝑆_𝑡)Rt+1​+𝛾V(St+1​)−V(St​)：TD error（误差）

𝑉(𝑆𝑡)←𝑉(𝑆𝑡)+α(𝑅𝑡+1+𝛾𝑉(𝑆𝑡+1)−𝑉(𝑆𝑡))𝑉(𝑆_𝑡) ← 𝑉(𝑆_𝑡) + \alpha(𝑅_{𝑡+1} + 𝛾𝑉(𝑆_{𝑡+1})-𝑉(𝑆_𝑡)) V(St​)←V(St​)+α(Rt+1​+𝛾V(St+1​)−V(St​))

  • 过程

    • 循环每个回合：
      • 初始化初始状态𝑆~0~
      • 𝑡 = 0,1,2, …:
        • 依据策略𝜋选择动作𝐴𝑡𝐴_𝑡At​，获得奖赏𝑅𝑡+1𝑅_{𝑡+1}Rt+1​和下一时刻状态𝑆𝑡+1𝑆_{𝑡+1}St+1​更新𝑉(𝑆𝑡)𝑉(𝑆_𝑡)V(St​)

  • 偏差方差的权衡

    • 回报𝐺𝑡𝐺_𝑡Gt​是𝑉𝜋(𝑆𝑡)𝑉^𝜋(𝑆_𝑡)V𝜋(St​) 的无偏估计
    • TD target 𝑅𝑡+1+𝛾𝑉(𝑆𝑡+1)𝑅_{𝑡+1} + 𝛾𝑉(𝑆_{𝑡+1})Rt+1​+𝛾V(St+1​)是𝑉𝜋(𝑆𝑡)𝑉^𝜋(𝑆_𝑡)V𝜋(St​)的有偏估计

  • 采样：TD和MC都有采样
