机器学习14：价值函数近似

发表于 2024-04-10 更新于 2024-04-15 分类于 ML
基于表格的方法将状态价值函数或动作价值函数储存到表格当中，并且需要学习每个状态下的价值，不仅消耗内存，同时单独的学习每个状态下的价值消耗时间较长，因此提出价值函数近似的方法，采用参数化的方式储存V、Q 、π...

价值函数近似

强化学习缺陷

  • 通过表格来记录价值函数
    • 状态价值函数：每个状态𝑠对应一个𝑉(𝑠)
    • 动作价值函数：每个(状态𝑠 - 动作𝑎)对应一个𝑞(𝑠, 𝑎)
  • 大规模MDP带来的困难
    • 存储空间大
    • 学习效率低

函数近似

  • 目标：将强化学习扩展至大规模的任务
  • 解决方案：函数近似
    • 通过函数近似来估计价值函数
      • 𝑣^(𝑠,𝒘)≈𝑉𝜋(𝑠)\hat 𝑣(𝑠,𝒘)≈𝑉^𝜋(𝑠)v^(s,w)≈V𝜋(s)
      • q^(𝑠,a,𝒘)≈q𝜋(𝑠,a)\hat q(𝑠,a,𝒘)≈q^𝜋(𝑠,a)q^​(s,a,w)≈q𝜋(s,a)
    • 从已经历的状态泛化到未经历的状态
    • 通过MC或者TD来更新参数𝒘
  • 常见函数近似器
    • 特征的线性组合(线性模型)
    • 神经网络

  • 无模型预测：评估策略的状态价值函数𝑉𝜋(𝑠)𝑉^𝜋(𝑠)V𝜋(s)

    • 实际上，无法得知真实的状态价值函数
    • 无模型预测：
      • 评估一给定策略的价值函数𝑉𝜋𝑉^𝜋V𝜋
      • 通过采样来更新𝑉𝜋𝑉^𝜋V𝜋 (MC或者TD)
        • MC: 往回报𝐺𝑡𝐺_𝑡Gt​更新
          • V(St)←V(St)+α(Gt−V(St))V(S_t)←V(S_t) + \alpha(G_t-V(S_t))V(St​)←V(St​)+α(Gt​−V(St​))
        • TD: 往TD target 𝑅𝑡+1+𝛾𝑉(𝑆𝑡+1)𝑅_{𝑡+1} + 𝛾𝑉(𝑆_{𝑡+1})Rt+1​+𝛾V(St+1​)更新
          • V(St)←V(St)+α(𝑅𝑡+1+𝛾𝑉(𝑆𝑡+1)−V(St))V(S_t)←V(S_t) + \alpha(𝑅_{𝑡+1} + 𝛾𝑉(𝑆_{𝑡+1})-V(S_t))V(St​)←V(St​)+α(Rt+1​+𝛾V(St+1​)−V(St​))
    • 无模型预测Vπ(s)V^\pi(s)Vπ(s) + 梯度下降学习𝑣^(𝑠,𝒘)\hat 𝑣(𝑠,𝒘)v^(s,w)
      • 在强化学习中，没有监督，只有反馈信号(奖赏)
      • 对目标𝑉𝜋(𝑠)𝑉^𝜋(𝑠)V𝜋(s)进行替代
        • MC: 使用回报𝐺𝑡𝐺_𝑡Gt​：
          • 𝐺𝑡𝐺_𝑡Gt​是V𝜋(𝑆𝑡)V^𝜋(𝑆_𝑡)V𝜋(St​)的无偏估计
          • 采样轨迹作为”训练数据“：<S1,G1>,...,<ST,GT><S_1,G_1>,...,<S_T,G_T><S1​,G1​>,...,<ST​,GT​>
          • 基于线性价值函数近似的MC预测：
            • Δw=𝛼(𝐺𝑡−𝑣^(𝑆𝑡,𝑤))∇𝑤𝑣^(S𝑡,𝑤)=𝛼(𝐺𝑡−𝑣^(𝑆𝑡,𝑤))x(St)\Delta w = 𝛼(𝐺_𝑡-\hat𝑣(𝑆_𝑡,𝑤))\nabla_𝑤\hat𝑣(S_𝑡,𝑤)=𝛼(𝐺_𝑡-\hat𝑣(𝑆_𝑡,𝑤))x(S_t)Δw=𝛼(Gt​−v^(St​,w))∇w​v^(St​,w)=𝛼(Gt​−v^(St​,w))x(St​)
            • 𝑤←𝑤+Δ𝑤𝑤←𝑤+\Delta 𝑤w←w+Δw
          • 在使用线性或者非线性价值近似时， MC预测总能收敛
        • TD: 使用TD target 𝑅𝑡+1+𝛾v^(𝑆𝑡+1,w)𝑅_{𝑡+1} + 𝛾\hat v(𝑆_{𝑡+1},w)Rt+1​+𝛾v^(St+1​,w)：
          • 𝑅𝑡+1+𝛾v^(𝑆𝑡+1,w)𝑅_{𝑡+1} + 𝛾\hat v(𝑆_{𝑡+1},w)Rt+1​+𝛾v^(St+1​,w)是V𝜋(𝑆𝑡)V^𝜋(𝑆_𝑡)V𝜋(St​)的有偏估计
          • 采样轨迹作为”训练数据“：<S1,𝑅2+𝛾v^(s2,w)>,<S2,𝑅3+𝛾v^(s3,w)>,...<S_1,𝑅_2+𝛾\hat v(s_2,w)>,<S_2,𝑅_3+𝛾\hat v(s_3,w)>,...<S1​,R2​+𝛾v^(s2​,w)>,<S2​,R3​+𝛾v^(s3​,w)>,...

        • Δw=𝛼(𝑅𝑡+1+𝛾maxa 𝑞^(𝑆𝑡+1,𝑎,𝑤)−q^(𝑆𝑡,At,w))x(St,At)\Delta w=𝛼(𝑅_{𝑡+1} + 𝛾\underset{a}{max}\ \hat 𝑞(𝑆_{𝑡+1}, 𝑎, 𝑤) -\hat q(𝑆_𝑡,A_t,w))x(S_t,A_t)Δw=𝛼(Rt+1​+𝛾amax​ q^​(St+1​,a,w)−q^​(St​,At​,w))x(St​,At​)

  • 并非利用目标函数的梯度进行更新

    包含两层近似：贝尔曼方程+价值函数近似
